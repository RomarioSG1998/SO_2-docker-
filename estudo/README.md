# Diretorio `estudo`

Este diretorio centraliza a documentacao tecnica do projeto.

## Funcao da pasta
- Registrar arquitetura, problemas encontrados e correcoes aplicadas.
- Servir como referencia rapida para manutencao e troubleshooting.

## Arquivos
- `arquitetura-projeto.md`: visao da arquitetura, fluxo de requisicoes, startup e dependencias entre servicos.
- `documentacao-erros.md`: historico de erros reais, causas e solucoes aplicadas durante a evolucao do ambiente.
- `resumo-correcoes.md`: resumo objetivo das principais correcoes para subir o projeto em maquina nova.
- `README.md`: esta documentacao da pasta.
